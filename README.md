# 🎓 QuizNova

QuizNova is a Full Stack Quiz Portal developed using React.js, Spring Boot, and PostgreSQL.

## 🚀 Features

- User Signup
- User Login
- Dashboard
- Java Quiz
- C Programming Quiz
- Python Quiz
- Operating System Quiz
- DBMS Quiz
- Data Structure Using C Quiz
- Quiz Result
- About Developer

## 🛠 Tech Stack

- React.js
- Spring Boot
- PostgreSQL
- REST API
- HTML
- CSS
- JavaScript

## 📂 Project Structure

```
QuizNova
│
├── frontend
│   └── quizapp
│
└── backend
    └── qapp
```
